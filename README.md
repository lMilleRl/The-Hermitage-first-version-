## About
My first attempt at creating a game. It never came to fruition due to its inherent complexity. Originally, the game was envisioned as a survival game with procedural world generation.

## Features
- Procedural world generation with Perlin Noise
- River generation based on Bézier curves
- Biome generation based on terrain elevation
- Chunk-based infinite world with LOD system
- Optimized mesh generation (Burst Compiler, Unity Jobs System)
  
## Tech Stack
Unity 6 | C# | Burst Compiler | Unity Jobs System | Physics3D
